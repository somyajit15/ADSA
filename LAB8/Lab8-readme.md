Fibonacci Heap Operation: Program demonstrates insertion, find minimum, and extract minimum operations using a Fibonacci Heap structure.

Splay Tree: Program implements insert, search, delete, and preorder traversal in a self-adjusting Binary Search Tree where accessed nodes are splayed to the root.

String Searching (KMP Algorithm): Program performs efficient pattern matching on a text using the Knuth–Morris–Pratt algorithm based on the LPS (prefix) table.
